This is a list of requirements for the queue file

the queue should take 8 bit width data
the queue should return 8 bit data
the queue should return data in the order it was receieved
the program using the queue should be able to have more than one copy of the queue
the queue should be able to return its size
the queue should not take more data than it can hold 
the queue should notify the caller on overflow
the queue should not return data if it is empty
the queue should notify the caller if overflow was attempted
the queue should be empty upon initialization
the queue should clear error status upon successful operation
